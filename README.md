# 🧟 Bob the Necromancer

> *"How did he die?"*

Bob wakes up with no memories. Only his name. A mysterious voice calling itself **Sam** echoes in his head, revealing one unsettling fact: there's a bounty on Bob's head. Navigate a dark, London-inspired open world as an unlikely necromancer, where every conversation you have and every choice you make reshapes the world around you.

---

## 🎮 About the Game

**Bob the Necromancer** is a top-down open-world RPG built in Java. It features a fully reactive world where your actions have real consequences, NPCs remember what you do, dialogue choices shift alliances, and no two playthroughs are quite the same.

* 🗺️ **Open world exploration**, roam freely and uncover the mystery of Bob's past
* 💬 **Consequence-driven dialogue**, every conversation leaves a mark on your world
* 🧠 **Reactive NPCs**, the world responds to who you are and what you've done
* ☠️ **Necromancy mechanics**, raise the dead, bend the rules of life

> ⚠️ **Early Prototype**, This project is in active early development. Expect bugs, missing features, and rapid changes.

## 🚀 How to Run

### Prerequisites

* Java JDK **17**

### Running the Game

1. Download and **unzip** the release package
2. Launch the game for your platform:
* **Windows**, double-click the `.bat` file
* **Linux / macOS**, double-click the `.sh` file *(open it first for instructions specific to your system)*

| Key | Action |
| --- | --- |
| `W` `A` `S` `D` | Move |
| `F` | Cast a spell |
